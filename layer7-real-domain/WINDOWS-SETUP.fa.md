## استفاده از V2RayN

### مرحله ۱: دانلود

https://github.com/2dust/v2rayN/releases

دانلود: `v2rayN-windows-64.zip`

### مرحله ۲: اجرا

فایل را extract کنید و `v2rayN.exe` را اجرا کنید.

### مرحله ۳: اضافه کردن سرور

#### روش A: اسکن QR Code
Servers → Scan QRcode from screen

#### روش B: Import Link
1. لینک VLESS/Trojan را کپی کنید
2. Servers → Import bulk URL from clipboard

### مرحله ۴: اتصال

روی سرور راست کلیک → **Set as active server**

✅ اتصال برقرار شد!
