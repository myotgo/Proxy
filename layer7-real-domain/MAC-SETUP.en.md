## Using V2RayX

### Step 1: Download

https://github.com/Cenmrev/V2RayX/releases

### Step 2: Install

Open DMG and copy to Applications.

### Step 3: Add Server

1. Open V2RayX
2. Configure → Import from JSON / QRCode
3. Paste VLESS/Trojan link or scan QR code

### Step 4: Connect

V2RayX icon in menu bar → **Load core**

✅ Connected successfully!

---

## Important Notes

- This method has the best security
- Uses real domain + valid TLS certificate
- Traffic looks like real HTTPS
