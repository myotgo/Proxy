## Using V2RayN

### Step 1: Download

https://github.com/2dust/v2rayN/releases

Download: `v2rayN-windows-64.zip`

### Step 2: Run

Extract and run `v2rayN.exe`.

### Step 3: Add Server

#### Method A: Scan QR Code
Servers → Scan QRcode from screen

#### Method B: Import Link
1. Copy VLESS/Trojan link
2. Servers → Import bulk URL from clipboard

### Step 4: Connect

Right-click server → **Set as active server**

✅ Connected successfully!

## Important Notes

- This method has the best security
- Uses real domain + valid TLS certificate
- Traffic looks like real HTTPS
- Keep UUID secure
