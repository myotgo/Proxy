## استفاده از V2RayX

### مرحله ۱: دانلود

https://github.com/Cenmrev/V2RayX/releases

### مرحله ۲: نصب

فایل DMG را باز کنید و به Applications کپی کنید.

### مرحله ۳: اضافه کردن سرور

1. V2RayX را باز کنید
2. Configure → Import from JSON / QRCode
3. لینک VLESS/Trojan را paste کنید یا QR Code را اسکن کنید

### مرحله ۴: اتصال

آیکن V2RayX در menu bar → **Load core**

✅ اتصال برقرار شد!
